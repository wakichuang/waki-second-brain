# 瓦基節錄評論法｜引導式提問庫

> 版本：v1.1
> 本檔案是「學習筆記專案包」的引導式提問範本庫。
> 階段 4 處理每一則筆記時，請從本檔案挑選最合適的提問搭配使用。

---

## 一、核心精神

**節錄評論法**是瓦基自創的學習筆記方法，源自《化輸入為輸出》課程。

它的核心信念是：

- **節錄**一段重點，**不只是抄寫**，而是要加上自己的觀點才有意義
- 評論不是憑空產生，而是**透過提問引導**，把使用者腦中的經驗、共鳴、質疑、應用想法挖出來
- 同一段重點，不同的人會給出不同的評論，這就是「卡片盒筆記」的靈魂

**這個提問庫的設計目的**：

不要讓使用者面對空白回答。每個提問都附帶 2 到 3 個方向選項，使用者可以「指認」一個方向，再自由補充細節。**指認比創造容易**，這是降低引導門檻的關鍵設計。

## 二、四大提問類型

每一則筆記的引導，從以下四種提問中挑選 1 到 2 個最合適的使用。

### 類型 1｜經驗連結型

**主提問**：

> 這段重點讓你想到自己過去的什麼經驗？

**為什麼這個提問重要**：

讓使用者把外在資訊跟個人記憶連結起來。**沒有個人經驗的卡片，讀起來像書摘，不像筆記。** 經驗連結是把書本內容變成「自己的東西」的第一步。

**方向選項範本**：

```
A. 工作或職涯中某個讓你印象深刻的專案或場景
B. 個人生活中某個習慣或行為（不論帶來好結果或壞結果）
C. 跟某位朋友 / 家人 / 同事的互動或對話
D. 其他（請自由分享你想到的連結）
```

**進階選項（用在已經對主題很熟的使用者）**：

```
A. 某個我做對了的決定，跟這段重點呼應
B. 某個我做錯了的決定，這段重點剛好幫我看見問題
C. 我正在掙扎中、還沒解決的某個情境
D. 其他（請自由補充）
```

---

### 類型 2｜共鳴探索型

**主提問**：

> 這段話為什麼讓你停下來？背後是什麼樣的共鳴？

**為什麼這個提問重要**：

當素材某段話讓使用者「畫線」、「停頓」、「想記下來」時，背後一定有某種共鳴在運作。把這個共鳴**說出來**，就是建立個人觀點的關鍵時刻。

**方向選項範本**：

```
A. 這段話講中了我一直想說、但說不清楚的東西
B. 這段話跟我曾經聽過 / 讀過的某個觀念互相呼應
C. 這段話幫我把過去某個模糊的經驗解釋清楚了
D. 其他（請自由分享共鳴的來源）
```

**進階選項（共鳴來自某個具體故事時）**：

```
A. 我聽過某個人的故事，跟這段重點很像
B. 我自己經歷過類似的事，當時感受很深
C. 我看過某部電影 / 影集 / 書，有類似的橋段
D. 其他（請自由分享）
```

---

### 類型 3｜觀點質疑型

**主提問**：

> 你哪裡不完全認同？你會怎麼補充或修正？

**為什麼這個提問重要**：

最有靈魂的學習筆記，是**有自己觀點**的筆記。完全照單全收只是抄書，部分質疑才是真正在思考。這個提問逼使用者跳出「作者說了就對」的被動模式。

**方向選項範本**：

```
A. 這段觀念在某個特定情境下不成立，例如 ⋯⋯
B. 我自己的經驗剛好相反，例如 ⋯⋯
C. 這段觀念忽略了某個重要因素，例如 ⋯⋯
D. 其他（請自由補充你的質疑或修正）
E. 我完全認同，請幫我做一次壓力測試
```

**如果使用者選 E**：

請你扮演「魔鬼代言人」，主動找出 2 到 3 個可能的反方論點，邀請使用者回應。例如：

> 好，我來壓力測試一下。我想到三個可能的反論：
>
> 1. 反論一：⋯⋯
> 2. 反論二：⋯⋯
> 3. 反論三：⋯⋯
>
> 你覺得哪一個最有挑戰性？或者你想怎麼回應這些反論？

---

### 類型 4｜未來應用型

**主提問**：

> 這段話未來可能在哪些地方派上用場？你會在什麼時刻想起它？

**為什麼這個提問重要**：

學習的目的是行動。沒有應用想像的筆記，三個月後一定會被遺忘。這個提問把抽象的觀念**轉換成具體場景**，讓使用者更容易記住、更容易實踐。

**方向選項範本**：

```
A. 我下次遇到 XXX 場景時可以用（例如：開會、做決定、面對衝突）
B. 我想分享給某個特定的人（家人、朋友、同事、學生）
C. 我會把這個觀念融入我的工作流程或個人習慣
D. 我會把它寫成自己的內容對外分享（文章、影片、簡報）
E. 其他（請自由補充應用場景）
```

**進階選項（深化「分享」這個方向）**：

```
A. 我會講給某位正在面對類似問題的朋友聽
B. 我會在下次團隊會議上分享給同事
C. 我會寫成一篇短文發在社群媒體
D. 我會收錄到我未來的演講 / 課程 / 內容裡
E. 其他（請自由補充）
```

## 三、提問選擇邏輯

不同類型的重點主題，適合搭配不同的提問。請依以下對照表挑選：

| 重點類型 | 優先提問 | 次要提問 |
|---|---|---|
| 方法論、技巧、SOP | 未來應用型 | 觀點質疑型 |
| 故事、案例、人物經歷 | 共鳴探索型 | 經驗連結型 |
| 觀點、主張、宣言 | 觀點質疑型 | 經驗連結型 |
| 數據、研究、實驗 | 觀點質疑型 | 未來應用型 |
| 心法、信念、哲學 | 共鳴探索型 | 未來應用型 |
| 比喻、隱喻、故事性敘述 | 經驗連結型 | 共鳴探索型 |

**選擇規則**：

1. **每則筆記至少用 1 個提問**，最多用 2 個提問
2. 如果這則筆記要用 2 個提問，**順序很重要**：先用優先提問建立基礎，再用次要提問深化
3. **不要每則都用同樣的提問**。同一輪對話中，盡量讓四種提問都出現過至少一次，讓使用者有不同的思考刺激

## 四、提問使用範例

以下是兩個完整範例，示範如何在實際對話中運用提問庫：

### 範例 1｜方法論類型的重點

**這則重點的核心**：「先動手做出版本一，再迭代修正，比想清楚再開始有效」

**挑選的提問**：未來應用型（優先）

**實際引導**：

```
〔引導提問〕

這個觀念講的是「先動手，再修正」。
未來在哪些地方你會想起它？

A. 我下次卡在「想清楚再開始」的時候可以用
B. 我想分享給也常常拖延的朋友 / 同事
C. 我會把它變成自己工作流程的固定原則
D. 我會寫成短文分享給其他人
E. 其他（請自由補充）

可以選一個，也可以選某個方向後再補充細節。
```

---

### 範例 2｜故事類型的重點

**這則重點的核心**：「某位作家每天清晨四點起床寫作三十年的故事」

**挑選的提問**：共鳴探索型（優先）+ 經驗連結型（次要）

**第一輪引導（共鳴探索）**：

```
〔引導提問 1〕

這個故事為什麼讓你停下來？

A. 我也想建立這種長期堅持的習慣
B. 這故事讓我反思自己的時間運用方式
C. 我認識某個朋友 / 家人也有類似的堅持，讓我很佩服
D. 其他（請自由分享共鳴的來源）
```

**第二輪引導（經驗連結，在使用者回答後追問）**：

```
〔引導提問 2〕

順著你剛剛說的方向，你過去有沒有類似「長期堅持某件事」的經驗？

A. 工作上某個我堅持下來、後來開花結果的事
B. 興趣或個人習慣中我一直在做的某件事
C. 我曾經放棄過、現在很後悔的某件事
D. 其他（請自由分享）
```

## 五、特殊情境的提問搭配

### 5.1 使用者完全沒想法時

如果使用者回答「我想不到」、「我不確定」：

- 不要再丟更多提問
- 改用「**選項收窄法**」：把原本的選項簡化成只有 2 個，外加 D 開放型
- 例如：「沒關係，那我把問題縮小一點。剛剛這段話讓你覺得：A. 很有道理，想記下來 / B. 有點疑惑，需要再想想 / C. 其他想法。」

### 5.2 使用者選了開放型 D 但回答很短

如果使用者選 D 但只寫了一兩個關鍵字：

- **追問細節**：「可以多說一點嗎？例如那是什麼樣的情境、發生在什麼時候、你當時的感覺如何？」
- 給 2 到 3 個「展開方向」的選項，例如：「你想多分享一下嗎？方向可以是：A. 那件事的細節經過 / B. 你當時的感受 / C. 那件事對你的影響。」

### 5.3 使用者偏離主題

如果使用者的回答跟這則筆記的核心觀念偏離：

- **不要否定使用者**
- 用一句話把使用者的回答「拉回主題」：「你提到的 XXX 很有意思，這跟我們這則重點的『YYY』可以怎麼連結？」
- 如果真的偏離太遠，誠實告訴使用者：「你說的這個方向可能更適合做成另一張卡片，我先記下來，等這則完成再回來處理，好嗎？」
