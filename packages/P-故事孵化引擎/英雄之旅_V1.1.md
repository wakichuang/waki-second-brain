# 英雄之旅 V1.1

## 一、英雄之旅是什麼？

英雄之旅（The Hero's Journey）是世界上最古老、也最普遍的故事原型。從荷馬史詩、希臘神話、好萊塢電影（《星際大戰》《魔戒》《哈利波特》），到無數人的真實人生故事，都可以套用這個結構。

最早是神話學家 Joseph Campbell 在《千面英雄》中提出 17 階段的英雄之旅。後來好萊塢編劇 Christopher Vogler 簡化為 12 階段。再後來，Donald Miller 在《跟誰行銷都成交》（StoryBrand）裡，把它**精簡為 7 個步驟**，並做了一個關鍵的翻轉：

> 「**你不是英雄。顧客才是英雄。你是引導者。**」

這個翻轉，讓英雄之旅從「**我的故事**」變成「**我幫對方完成的旅程**」——這就是為什麼 StoryBrand 的版本特別適合**品牌故事、商業敘事、人生大轉變故事**。

---

## 二、為什麼選 StoryBrand 七步驟，而不是 Campbell 17 階段？

兩個原因：

**1. 簡單，可以實際用**
17 階段太細，多數人寫不完。7 步驟剛剛好，能完整呈現英雄之旅的精髓，又能在 30 分鐘內完成。

**2. 對焦中文世界的真實應用**
StoryBrand 在中文世界已經有翻譯版本，瓦基本人在 readingoutpost.com/building-a-storybrand 寫過完整書評，並親自把這套方法用在自己網站首頁的改造上。等於有實作案例可以參考。

> 瓦基書評金句：「**改造前我把自己當成英雄，急著展示我有什麼特色。改造後我才驚覺『顧客才是英雄』。我站在讀者的角度去走一趟英雄之旅，當我發揮同理、感受對方遭遇的問題的時候，寫下來的文字才終於有了點模樣。**」

---

## 三、StoryBrand 七步驟總覽

| 步驟 | 名稱 | 核心問題 |
|---|---|---|
| 1 | **角色（英雄）** | 主角是誰？他想要什麼？ |
| 2 | **問題** | 他遇到了什麼問題？外在、內在、哲學三層 |
| 3 | **引導者** | 誰幫他？引導者擁有什麼能力？ |
| 4 | **計畫** | 引導者給了他什麼計畫 / 方法？ |
| 5 | **行動呼籲** | 他被推動採取什麼行動？ |
| 6 | **避免失敗** | 如果他不行動，會失去什麼？ |
| 7 | **達到成功** | 他最終獲得了什麼？變成什麼樣子？ |

這七個步驟可以套用在兩種情境：

- **品牌 / 商業故事**：顧客是英雄、你是引導者
- **個人大轉變故事**：你是英雄、你遇到的某人 / 某本書 / 某個契機是引導者

## 四、七步驟詳細展開

### 步驟 1｜角色（英雄）

**核心問題**：故事的主角是誰？他想要什麼？

關鍵：**清楚定義「想要」**。

如果英雄沒有明確的「想要」，故事就沒有方向。

範例：
- 品牌故事：「我服務的對象，是 30-45 歲的知識工作者，他們**想要**用 AI 工具產出有品質的內容。」
- 個人故事：「我那時候是台積電工程師，我**想要**找到一份能讓我每天起床覺得有意義的工作。」

**寫作技巧**：
- 不要寫「英雄很厲害」，要寫「英雄想要什麼」
- 「想要」要具體（不是「想要成功」，是「想要每天起床覺得有意義」）
- 「想要」要連結到一個**情感層面**（不只是行動目標）

### 步驟 2｜問題（三層問題）

**核心問題**：英雄遇到了什麼**阻礙**他達成想要？

StoryBrand 強調，問題有**三個層次**——多數人只寫第一層，其實第二、第三層才是讓故事打動人的關鍵：

**第一層：外在問題（External）**
具體、表面、可被觀察的問題
範例：他不知道怎麼用 ChatGPT 寫出像樣的文章。

**第二層：內在問題（Internal）**
英雄因為外在問題產生的**感受**
範例：他每次跟 AI 對話完都覺得「我是不是太笨？」「為什麼別人用 AI 用得那麼順？」

**第三層：哲學問題（Philosophical）**
這個問題背後**更大的不公平 / 不該如此**
範例：「為什麼工具明明就在那裡，卻好像有人天生會用、有人怎麼學都學不會？AI 時代不應該是這樣的。」

> **三層問題的力量**：當你只寫外在問題，對方覺得「跟我無關」。當你寫到內在問題，對方覺得「這就是在說我」。當你寫到哲學問題，對方覺得「這個世界應該被改變」。

### 步驟 3｜引導者

**核心問題**：誰幫助英雄？引導者擁有什麼能力？

關鍵：**引導者要展現「同理 + 權威」**。

- **同理**：「我懂你的處境，我也曾經這樣」
- **權威**：「但我找到了方法 / 工具 / 路徑，我可以帶你走過」

範例：
- 品牌故事：「我自己也曾經從工程師轉行做創作者。過去 5 年，我累積超過 200 萬字、開過 3 堂線上課。我知道從零到一的痛，也知道怎麼走過來。」
- 個人故事：「直到我遇到了 [那個人 / 那本書 / 那個契機]。他 / 它 / 那件事讓我看到了不一樣的可能。」

**寫作技巧**：
- 引導者**不是主角**——他出現是為了讓英雄繼續前進
- 同理要在權威之前——先讓對方覺得「你懂我」，再讓對方相信「你能幫我」
- 引導者的「權威」不是靠自吹自擂，是靠**具體證據**（經歷、數字、作品）

### 步驟 4｜計畫

**核心問題**：引導者給了英雄什麼**具體的方法或步驟**？

人類面對未知會緊張。**計畫的作用是降低焦慮**——讓英雄看見「我只要這樣做、再這樣做、最後這樣做」就可以走到終點。

計畫有兩種寫法：

**A. 步驟式計畫**（適合教學、產品、服務）：
1. 第一步：[具體動作]
2. 第二步：[具體動作]
3. 第三步：[具體動作]

**B. 承諾式計畫**（適合品牌、價值主張）：
「我承諾你：[一句明確的承諾]，前提是你願意 [簡單的條件]。」

範例：
- 「跟我學三件事就夠了：(1) 學會跟 AI 對話的鐵則 (2) 建立你的個人風格指南 (3) 用 PACE 四能力組合提示詞。」
- 「我承諾你 12 週後能產出第一份個人作品，前提是你願意每週花 3 小時走完一個專案包。」

### 步驟 5｜行動呼籲

**核心問題**：英雄被推動採取**什麼具體行動**？

行動呼籲分兩種：

**A. 直接行動呼籲（Direct）**：明確的「現在就做」
範例：「點此立即加入課程」「現在就報名」「下載免費電子書」

**B. 過渡行動呼籲（Transitional）**：先建立關係，再推動購買
範例：「先訂閱電子報，每週收到一篇思考筆記」「先免費試用 7 天」

關鍵：**故事不能停在「英雄想要」就結束**。讀者需要被推動「**現在做什麼**」。

### 步驟 6｜避免失敗（什麼會失去）

**核心問題**：如果英雄不行動，他會**失去什麼**？

人類的決策受**損失趨避**驅動——比起「得到什麼」，「失去什麼」更能推動行動。

範例：
- 「如果你繼續用現在這種方式跟 AI 對話，5 年後別人會用 AI 產出 100 倍的價值，你會被甩在後面。」
- 「如果你不在這次轉折點做出選擇，下一次可能要再等 10 年。」

**寫作技巧**：
- 不要販賣恐慌（這違背瓦基品牌調性）
- 但要誠實揭露「不行動的真實代價」
- 損失要具體（時間、機會、成長、關係），不要抽象

### 步驟 7｜達到成功

**核心問題**：英雄最終獲得了什麼？變成什麼樣子？

這是故事的**新常態**——對應 Kindra Hall 三幕劇的第三幕。

成功有三個層次（呼應步驟 2 的三層問題）：

**第一層：外在的成功**
「他學會了用 AI 寫出有品質的文章。」

**第二層：內在的成功**
「他開始相信『我也是 AI 時代的贏家』。」

**第三層：哲學的成功**
「他不只解決了自己的問題，還能幫其他人走過同樣的路。」

**寫作技巧**：
- 三層都要寫到，故事的張力才會完整
- 成功要可被「具體想像」（讀者腦中能浮現畫面）
- 成功不一定要圓滿，但要呼應步驟 1 的「想要」

## 五、StoryBrand 七步驟的兩種應用方式

### 應用 A｜品牌故事（顧客是英雄）

這是 StoryBrand 最原始的用法。把英雄、引導者的角色明確分配：

```
英雄 = 你的顧客 / 服務對象
引導者 = 你 / 你的品牌
```

完整骨架：

> 「你是一個 [步驟 1：顧客的角色與想要]。
>
> 你正在面對 [步驟 2：外在 + 內在 + 哲學三層問題]。
>
> 我是 [步驟 3：引導者身分]。我自己也曾經 [同理]，後來我找到了 [權威]。
>
> 我可以提供你 [步驟 4：計畫]。
>
> [步驟 5：現在開始 / 先免費體驗]。
>
> 如果你不行動，[步驟 6：你會失去什麼]。
>
> 跟我走完這條路，你會 [步驟 7：達到的三層成功]。」

### 應用 B｜個人大轉變故事（你是英雄）

把英雄留給自己，引導者讓給「曾經幫助你的某人 / 某事 / 某本書」：

```
英雄 = 你（過去的自己）
引導者 = 那個關鍵的人、事、書、契機
```

完整骨架：

> 「我那時候是 [步驟 1：我的身分與想要]。
>
> 我面對的是 [步驟 2：三層問題]。
>
> 直到我遇到了 [步驟 3：引導者 / 關鍵契機]。它 / 他 / 那件事讓我看到 [權威]。
>
> 我開始 [步驟 4：採取的方法或路徑]。
>
> [步驟 5：我做出的具體行動]。
>
> 如果當時我沒有做出選擇，[步驟 6：我會失去什麼]。
>
> 走到現在，我 [步驟 7：達到的三層成功]。」

**判斷使用 A 還是 B**：

| 你的故事目的 | 用 A 還是 B |
|---|---|
| 對外的品牌敘述、行銷文案 | A（顧客是英雄） |
| 我的職涯轉換故事 | B（我是英雄） |
| 服務頁、課程介紹 | A |
| 自傳、人生回顧 | B |
| 創業歷程（對潛在投資人） | 混合：以 A 為主，穿插 B |

## 六、英雄之旅的兩種寫作風格選擇

寫完七步驟之後，可以選擇用兩種風格呈現：

### 風格 1｜直敘式（適合品牌頁、商業文案）

按七步驟順序，一段一段寫。讀者明確看到「英雄→問題→引導者→計畫→行動→失敗→成功」的結構。

**優點**：邏輯清楚、易於閱讀、適合行銷情境
**缺點**：故事感較弱，比較像「結構化說明」

### 風格 2｜敘事式（適合個人故事、自傳、Podcast 開場）

不按七步驟順序，從某個**強烈的場景**開始，倒敘或交錯展開，讓七步驟的元素自然散落在故事中。

**優點**：故事感強、情感層次豐富、適合長文或口語敘述
**缺點**：寫作難度較高、需要更多素材

## 七、英雄之旅檢核清單

每次寫完一份故事，過這份清單：

**步驟完整性**：
- [ ] 步驟 1：英雄是誰？他「想要」什麼是否具體？
- [ ] 步驟 2：三層問題（外在、內在、哲學）是否都有寫到？
- [ ] 步驟 3：引導者是否同時展現了「同理」與「權威」？
- [ ] 步驟 4：計畫是否具體？讀者能否清楚看到下一步？
- [ ] 步驟 5：行動呼籲是否明確？
- [ ] 步驟 6：不行動的代價是否誠實揭露（而非販賣恐慌）？
- [ ] 步驟 7：成功是否寫到三層（外在、內在、哲學）？

**整體張力**：
- [ ] 英雄與引導者的角色分配是否清楚？（A 應用：顧客是英雄；B 應用：我是英雄）
- [ ] 開頭與結尾是否呼應？
- [ ] 讀者讀完之後，知道自己「現在要做什麼」嗎？

十題裡有任何一題答不出來，這個故事就還沒準備好。

## 八、適用情境總表

| 情境 | 是否適用英雄之旅 |
|---|---|
| 品牌官網的服務介紹 | ✅ 用應用 A |
| 個人職涯大轉變 | ✅ 用應用 B |
| 創業歷程敘述 | ✅ 用應用 B |
| 自傳 / 人生回顧 | ✅ 用應用 B |
| 課程銷售頁 | ✅ 用應用 A |
| 對潛在客戶的提案 | ✅ 用應用 A |
| 短篇個人小故事（單一事件） | ⚠️ 改用弧形結構 |
| 品牌的某個獨立故事（顧客見證） | ⚠️ 改用三幕劇 + 四種故事 |
| 跨越好幾年的成長故事 | ✅ 用應用 B |

## 九、一個重要的提醒

英雄之旅這個結構很強大，但**強大的工具也容易被濫用**。請注意三件事：

**1. 不要把每個故事都套成英雄之旅**
有些小故事用弧形結構就夠了。硬要套七步驟反而會讓故事變得「公式化」。

**2. 不要過度英雄化自己**
應用 B 寫個人故事時，要小心不要把自己描繪成「神奇地戰勝所有困難」的英雄。**真實的脆弱**比英雄式的完美更動人。

**3. 不要把「失敗」寫成恐嚇**
步驟 6 是「誠實揭露代價」，不是「販賣焦慮」。語氣要溫和，事實要清楚。
