# 三幕劇與四種故事 V1.3

## 一、為什麼有些故事讓人記住、有些故事看完就忘？

Kindra Hall 是美國知名的說故事教練，她創立的 Steller Collective 顧問公司專門幫企業用故事跟客戶溝通。她長期觀察到一個現象：

> 為什麼有些公司難以獲得客戶青睞，必須打規格戰、價格戰？有些公司卻擁有大量鐵粉，價值與形象總是深植人心？

她發現差別在於兩件事：

1. 他們**是否使用故事**跟對象溝通
2. 他們**是否掌握說故事的能力**

這個觀察不只適用於企業，也適用於個人品牌、創業者、簡報者、任何想要「**讓對方記住你**」的人。

這份檔案就是教你：用**三幕劇結構**寫好故事，並用**四種故事類型**找到你該說的故事。

---

## 二、故事橋的概念

Kindra Hall 提出一個核心隱喻：故事是一座**橋**。

任何溝通的起點是 A（你 / 你的品牌），終點是 B（你的對象 / 你的顧客）。**A 跟 B 之間原本有一條鴻溝**：

- 對方不認識你
- 對方不知道為什麼要關注你
- 對方有抗拒、有懷疑

要跨越這條鴻溝，你需要一座**故事橋**。

故事橋的三個關鍵作用：

1. **引起注意力**：捕捉對方的注意力，讓他願意聽你說
2. **發揮影響力**：影響對方的感受，讓他放下抗拒
3. **引發長期改變**：讓對方從內心產生變化，不會走回橋的另一頭

> 「說故事的力量，就是當其他辦法都宣告失敗的時候，只有故事還是能搭橋跨越鴻溝。」
> ——《牧羊少年奇幻之旅》作者，保羅．科爾賀

## 三、三幕劇結構

Kindra Hall 歸納她擔任故事教練多年的經驗，認為**最生動又有效**的故事架構就是三幕劇：

| 幕 | 名稱 | 核心問題 |
|---|---|---|
| 第一幕 | **常態** | 事情發生**之前**，世界是什麼樣子？ |
| 第二幕 | **爆炸性發展** | 發生了什麼？打破了常態？ |
| 第三幕 | **新常態** | 事情過後，世界變成什麼樣子？ |

### 第一幕｜常態（最被低估、最重要的一幕）

**Kindra Hall 特別強調**：常態是故事**最重要**的部分。

為什麼？因為：
- 沒有常態，讀者不知道「正常」長什麼樣子
- 沒有常態，讀者就不會在乎「打破常態」這件事
- 常態是讀者「跟你站在同一起點」的基礎

**寫好常態的關鍵**：
- 加入細節（具體的場景、人物、日常）
- 建立熟悉感（讓讀者覺得「我也經歷過類似的」）
- 讓聽眾**有理由在乎**你的故事

### 第二幕｜爆炸性發展

> 「局勢的架構突然『發生變化』，**變好或變壞其實不重要**。重點是發生了某件事，導致新常態的產生。」

注意這句：「**變好或變壞其實不重要**」。

很多人以為故事的中段一定要是「困難、危機、挫敗」。其實不是。爆炸性發展也可以是：

- 一次意外的機會
- 一次令你驚訝的相遇
- 一次無預警的變動（不一定是壞事）
- 一個你原本不相信的事實，突然出現在眼前

**重點不在於正負，而在於「改變」**。

### 第三幕｜新常態

> 「局勢在『爆炸』發生後變成什麼模樣。這個步驟要讓『故事』能有效傳達某個重點，或強化某個訊息。」

新常態是故事的**目的地**。它要承載：

- 故事的領悟、訊息、價值
- 讀者「為什麼要聽這個故事」的答案
- 你想要對方在故事結束後**帶走的東西**

**重要**：新常態必須跟第一幕的常態形成「**對比**」，讓讀者感受到「世界真的不一樣了」。

## 四、精彩故事的四個必備環節

光有三幕劇還不夠。Kindra Hall 認為，一個訊息如果擁有特定故事環節，會比缺乏這些環節的訊息更吸引人。

### 環節 1｜鮮明角色

> 「我們需要的不是英雄，而是鮮明的角色，我們在乎而且同情的對象。」

注意：**不需要是英雄**。讀者不需要崇拜主角，只需要**在乎**他、**同情**他。

讓角色鮮明的方法：
- 給他一個具體的外觀特徵（不是「她很美」，是「她那天穿了一件灰色毛衣」）
- 給他一個習慣動作（「他講話時總會抓後腦勺」）
- 給他一句口頭禪（「他每次提到媽媽都會說『欸我跟你說』」）

### 環節 2｜真實情感

> 「情感指的並不是聆聽者本身的體驗，而是故事角色的感受，或本來就存在於故事的情境。透過這種情感，聽者對故事產生同理心。」

情感不是寫給讀者「應該感受到什麼」，是把「故事人物當下的真實情緒」描寫出來，讓讀者**自己決定要不要感同身受**。

寫情感的關鍵：**用身體反應、行為、對話，而非情緒形容詞**。

範例：
- ❌ 「他很憤怒。」
- ✅ 「他把咖啡杯放下，但放得太用力，杯子發出『叩』一聲。他沒看我，只是看著桌面。」

### 環節 3｜特定時刻

> 「在空間、時間或情境中的特定一個點。把故事放在一個方便湊近觀看的位置，能放大原本平凡無奇的敘述，讓觀眾將其中一幕看得更清楚。」

故事不要寫「**一段時間**」，要寫「**那一刻**」。

範例：
- ❌ 「那段時間我們的關係變得很僵。」
- ✅ 「那是星期六的晚餐桌上。我問他週末要不要去看電影，他連頭都沒抬，只說『我累了』。那是三個月來他第一次拒絕我。」

特定時刻**像一個鏡頭聚焦點**——把整段故事的張力，濃縮到某一個畫面裡。

### 環節 4｜具體細節

> 「越具體越好，細節用得越多，聽眾就越能感受到你多麼了解他們，進一步產生共鳴的感受。」

具體細節的作用是「**證明這是真的**」。

四個層次的具體：
1. **時間**：年份、季節、星期幾、白天還是晚上
2. **地點**：城市、區域、店名、街角
3. **人物**：姓名、年紀、外觀、跟你的關係
4. **物件**：他手裡拿的東西、桌上的擺設、身上的氣味

**金句**：細節用得越多，聽眾就越能感受到你**多麼了解他們**。

## 五、四種故事類型（每個人 / 每個品牌都該說的故事）

Kindra Hall 提出四種故事類型，分別服務不同的溝通目的。

### 類型 1｜價值故事

**目的**：提升銷售與行銷成果

**核心**：讓對方意識到「**沒錯，這就是我需要的解決方案**」

**標準結構**：
1. 現有顧客或潛在顧客面臨著麻煩與問題
2. 他們正在掙扎、想辦法應付、判斷有沒有更好的方法
3. 你的產品或服務出現了
4. 顧客使用了你的產品或服務
5. 他們的人生變得更美好，問題獲得解決

**適用情境**：
- 銷售頁、產品介紹頁
- 提案、提案開場
- 課程介紹、服務介紹
- 為某個產品 / 服務寫一篇推薦文

> 「行銷的重點不再是你做了什麼東西，而是你說了什麼故事。」
> ——行銷專家，賽斯．高汀

### 類型 2｜創辦人故事

**目的**：提高顧客、員工、投資者對公司的信心

**核心**：差異化。當每個人都用「追求卓越、客戶服務、熱忱」這些標語時，**你的與眾不同，就只是跟別人一樣**。創辦人自己的故事，才能帶來真正的差異化。

**標準結構**：
1. 創辦人原本的生活、職業、想法
2. 一個關鍵的觸發點（一次失敗、一次發現、一次轉變）
3. 創辦人的選擇與行動
4. 創業過程的真實情感、靈光乍現、血汗時刻
5. 現在的公司或品牌長什麼樣子，跟最初的承諾如何呼應

**適用情境**：
- 個人品牌介紹頁、關於我頁面
- 公司官網的創辦人介紹
- 募資簡報的開場
- 媒體採訪的自我介紹

> 「如果你投資的某人根本不相信自己的故事，你又怎麼會相信人家？」
> ——社會心理學家，艾美．柯蒂

### 類型 3｜使命故事

**目的**：凝聚團隊和組織成員，激勵人心

**核心**：是人，就會想要使命。當個人和團隊為了一件有「使命」和「意義」的事情奮鬥，向心力和執行力就會更上一層樓。

**標準結構**：
1. 為什麼這件事重要（不是「我們做了什麼」，是「我們為什麼做」）
2. 一個具體的故事或場景，呈現這個「為什麼」
3. 我們的承諾與行動
4. 這份使命對團隊 / 對社會的意義

呼應賽門．西奈克的「**黃金圈理論**」：
- 內圈：為什麼（Why）← 使命故事服務這一圈
- 中圈：怎麼做（How）
- 外圈：做什麼（What）

**適用情境**：
- 公司使命宣言、企業文化
- 對團隊的演講
- 招募新人時的價值觀說明
- 對外溝通品牌精神

### 類型 4｜顧客故事

**目的**：改善銷售與行銷成果

**核心**：可信度。我們會懷疑賣家的話，但會相信「跟我一樣的普通人」說的話。

**標準結構**：
1. 顧客原本的狀態與問題
2. 他們遇到你的產品 / 服務的契機
3. 使用過程中發生的事（最好有具體場景、對話、細節）
4. 結果：他們的人生 / 工作 / 生活如何改變

**關鍵兩件事**：
1. **你必須主動開口請求**顧客分享他們的故事
2. **請求要夠詳細**才能獲得所求：你希望顧客做出什麼樣的回應，就該對顧客提出什麼樣的疑問

**適用情境**：
- 顧客見證、推薦文
- 案例研究（Case Study）
- 行銷貼文（分享別人的成功故事）
- 銷售過程中的舉例

> 「所謂的品牌，是人們在你不在場的時候如何評價你。」
> ——亞馬遜執行長，傑夫．貝佐斯

## 六、四種故事的決策樹

當你不確定該說哪一種故事時，問自己：

| 你的目的 | 該說的故事類型 |
|---|---|
| 我想賣出某個產品 / 服務 | **價值故事** |
| 我想讓別人對我 / 我的公司產生信任 | **創辦人故事** |
| 我想凝聚團隊或對外傳達使命 | **使命故事** |
| 我想提高可信度、用第三方視角證明價值 | **顧客故事** |

## 七、三幕劇與四環節的綜合檢核清單

每次寫完一份故事，過這份清單：

**三幕劇結構**：
- [ ] 第一幕：我有沒有花夠多篇幅建立「常態」？讓讀者跟我站在同一起點？
- [ ] 第二幕：爆炸性發展有沒有被「**寫慢**」？讀者有沒有感受到改變的衝擊？
- [ ] 第三幕：新常態有沒有跟第一幕形成對比？讀者有沒有得到他要帶走的東西？

**四環節檢核**：
- [ ] **鮮明角色**：故事中的人物有沒有「具體的外觀 / 動作 / 語氣」？
- [ ] **真實情感**：我有沒有用身體反應、行為、對話寫情感，而非情緒形容詞？
- [ ] **特定時刻**：故事的高峰有沒有聚焦在「那一刻」，而不是「那段時間」？
- [ ] **具體細節**：時間、地點、人物、物件，有沒有給足細節？

七題裡有任何一題答不出來，這個故事就還沒準備好。

## 八、適用情境總表

| 情境 | 適合的故事類型 |
|---|---|
| 個人 / 公司的對外品牌建立 | 創辦人故事 |
| 銷售產品、推銷服務 | 價值故事 |
| 內部團隊凝聚、文化宣導 | 使命故事 |
| 客戶見證、推薦文 | 顧客故事 |
| 募資簡報的開場 | 創辦人故事 + 使命故事 |
| 個人品牌的關於我頁面 | 創辦人故事 |
| 課程銷售頁 | 價值故事 + 顧客故事 |
| 給員工的演講 | 使命故事 |

## 九、最後一個提醒：你比你以為的更有故事

> 「阻止你說故事的最大障礙，不是拖延、害怕分享，也不是怯場，而是你從一開始就認定自己沒故事可說。」

很多人覺得自己「只是個普通人，哪有什麼故事好說」。

但 Kindra Hall 強調：「**只要故事是真的，無論多大多小、是悲是喜，聽眾一定會產生共鳴。**」

你不需要驚天動地，只需要**真實**。
