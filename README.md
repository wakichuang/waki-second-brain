# AI 瓦基第二大腦｜課程專案包資源庫

這個倉庫存放「AI 瓦基第二大腦」線上課程的專案包原始檔，並負責版本管理與學員下載。

每個專案包都是一組可以部署到 ChatGPT GPTs、Gemini Gems 或 Claude Project 的設定，分成兩個部分：

- **Instructions（指令）**：定義這個專案包的行為。
- **知識庫（Knowledge）**：放方法論與參考資料。

## 資料夾結構

```
Waki-second-brain/
├── README.md              ← 你正在看的這份
├── CLAUDE.md              ← 給 Claude Code 的維護與更新規則
├── index.html            ← 學員下載儀表板（landing page）
├── downloads/            ← 打包好的下載檔
│   ├── 專案包全集.zip        ← 一鍵下載全部（所有專案包）
│   └── <專案包>_原始檔.zip    ← 各專案包的原始檔打包（卡片「下載原始檔」用）
└── packages/             ← 所有專案包的原始檔
    └── <分類>-<名稱>/
        ├── Instructions 檔（沿用原始檔名，可能有完整版與 GPTs 版）
        └── 知識庫各檔（沿用原始檔名）
```

## 專案包清單（依 PACE 分類）

**P　Perspective：觀點打造**

- 觀點作品引擎：把模糊的想法挖成清楚的觀點，一路寫成文章、Email、提案或說服訊息。
- 故事孵化引擎：把你的回憶，孵化成扣人心絃的個人故事。

**A　Awareness：自我覺察**

- 思考覺察日記：情緒整理、日記引導，回頭看自己的內在。
- 人際關係對話陪練：困難對話先預演，衝突過後做復盤。

**C　Self-Correction：自我校正**

- 決策分析引擎：把難下的決定，拆成清楚的選項、取捨與盲點。
- 策略規劃引擎：從目標倒推，用金字塔結構排出可執行的計畫。
- 簡報骨架引擎：用 SCQA 開場、金字塔展開，先把簡報的骨架立起來。

**E　Empowerment：賦能**

- 智囊團：8 到 10 種不同立場的角色，輪流挑戰你的觀點。
- 洞見引擎：沒靈感的時候，幫你往外發散、把想法的維度拉高。
- 學習筆記：書籍、課程、影音的統整與筆記，內含多源餵料技巧。

**進階工具**

- 寫作風格萃取器：萃取出你自己的寫作風格，變成可以餵給 AI 的風格指南。
- 專案包建造器：用一問一答的方式，帶你從零打造一個新的專案包。
- 查證引擎：幫你查證文章裡的來源、數據與引用，揪出錯誤、AI 幻覺與張冠李戴。

## 如何更新

要更新某個專案包到新版本、或是新增專案包時，請依照 [CLAUDE.md](CLAUDE.md) 裡的「更新某個專案包的標準流程」進行。流程會自動保留舊版本、重新打包，並同步更新儀表板的資料。
